Declaración de Accesibilidad

En [Nombre del Sitio], nos esforzamos por garantizar que nuestro sitio web sea accesible para todas las personas, independientemente de sus capacidades y tecnología utilizada para acceder a la web.

Hemos tomado medidas para hacer que este sitio web sea fácilmente navegable y utilizable para personas con discapacidades, incluidas aquellas que dependen de tecnologías de asistencia como lectores de pantalla. Nuestro objetivo es cumplir con las pautas de accesibilidad web establecidas por el World Wide Web Consortium (W3C) en sus Directrices de Accesibilidad para el Contenido Web (WCAG) 2.1, nivel AA.

Si tiene alguna dificultad para acceder a la información en este sitio web o tiene sugerencias sobre cómo podemos mejorar la accesibilidad, no dude en ponerse en contacto con nosotros.
